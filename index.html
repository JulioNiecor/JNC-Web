Sobre mí

¡Hola! Soy Julio, un apasionado desarrollador web junior con experiencia en HTML5, CSS, JavaScript y Angular, además de habilidades en GitHub. Mi enfoque autodidacta me impulsa a embarcarme en proyectos desafiantes para aprender continuamente.

Mi filosofía es abordar cada línea de código como una oportunidad de crecimiento. Busco no solo cumplir, sino superar expectativas. Si buscas un profesional comprometido y entusiasta, mi portfolio cuenta la historia visual de mi viaje y habilidades. ¡Espero entusiasmar a tu equipo con mi contribución!

Proyectos

TODO App

Este proyecto es una implementación simple de una lista de tareas interactiva utilizando HTML, CSS y JavaScript, con la incorporación del framework Bootstrap para mejorar la interfaz de usuario. La aplicación permite al usuario agregar, editar, marcar como completadas y eliminar tareas de manera dinámica.

CineFlix

Este proyecto es una aplicación web desarrollada en Angular que permite a los usuarios explorar sus películas y series favoritas. Utiliza la API de The Movie Database (TMDB) para obtener información detallada sobre películas y series, y ofrece funcionalidades como búsqueda y visualización de detalles.
